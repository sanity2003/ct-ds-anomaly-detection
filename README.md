# CT-DS Hiring April Challenge

## Overview
This project was developed as part of the Celebal Technology CT-DS Hiring Challenge. The objective was to detect anomalies in industrial sensor data using machine learning techniques.

## Achievement
- Rank: 70/200+
- Best Competition Score: 0.7032
- Dataset Size: 1.6M+ records

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- LightGBM
- XGBoost
- SMOTE

## Project Workflow
1. Data Loading & Validation
2. Exploratory Data Analysis
3. Feature Engineering
4. Class Imbalance Handling using SMOTE
5. Model Training (LightGBM, XGBoost)
6. Threshold Optimization
7. Prediction & Evaluation

## Key Results
- Processed 1.6M+ records
- Improved anomaly detection performance through class balancing
- Tuned model thresholds to optimize F1-score
